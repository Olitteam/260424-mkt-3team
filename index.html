        • 현재 시트의 “공헌이익률”이 마케팅팀 목표가 되는게 맞는지?
          • CRM, 반품, 취소, 물류, 원가 > 기타 비용
          • 마케팅과 매출 > ROAS = 구매율 + CTR
        • 데이터가 신빙성이 없는게 가장 크고
          • 제품별 의사 결정에 올릿 상세 매출 보고소 슬룸 의 데이터가 오히려 잘못된 정보
          • 예시 1: V1 공헌이익만 개선 플레이가 어려운 현 상황
            • 혼합 페이지
              • 상황:
                • V1 성과 개선 > V1 콘텐츠 제작 > V1 광고비 상승
                  • V1 매출 유지 OR 소폭 상승 / V2 매출&성과 상승
              • V1,V2 판매 페이지 > 해당 페이지 V1 구매율 1% / V2 구매율 3%
              • 결론: V1 성과 하락, V2 성과 상승
                • 해당 지표는 액션의 근거로 사용되기 어려움
                • 현재 [발편한케어] 상황에는 [V1 별도 공헌이익]은 맞지 않는 목표 수치
            • 혼합 캠페인
              • V1,V2 광고 혼용 운용
                • 랜딩이 2개인 캠페인 OR 세트 [예산 운용 관련]
                  • V1 지출로 잡히고 V2 효율이 좋아짐
                  • V1 데이터 성과 체크 후 예산 중단 > V2효율도 안 좋아짐
          • 예시 2: 허리편한케어
            • 3월 가상 계산
            • 3월 기타 비용을 제외하고 보면 성과 좋음
              • GFA 1일 50만원 > 스마트스토어 매출 근거
              • CRM > 1일 최고 비용 03/09 > 225만원 / 당일 공헌 이익 - 119만원
            • 4월에 진행된 액션들이 하지 않아도 되는 액션들이라 판단
              • 허리편한케어 V1 페이지 개선
                • 1월, 3월 V1 페이지 구매율 2% 이상 / 2월 중단 사유 모름
                  • V1 광고 흰색 콘텐츠 280개 / 파란색 콘텐츠(PA) 7개
                • 4월 6일 V1 페이지 변경 후 구매율 0.68%

          • 발편한케어 V2 판매 촉진

            • 해결안: V1 단독 페이지 or V1+V2 페이지 개선 전까지 V2 집중

            • 1. 파란색 제품으로 촬영부터 필요 [광고 콘텐츠 부재] / 발편한케어와 상황 다름

            • 자사몰 129,000원 VS 스마트스토어 109,000원 + 네이버 광고 진행
          • D2C팀 - 프로모션
            • V1[129,000원]은 4월 27일 프로모션, 번들 X
            • V2[129,000원]는 프로모션 상단, 번들 구성 10가지

            • V2+코어요추벨트[119,900원]

                  • 광고 세팅 [채널 다각화, 광고비 운영 효율화]
